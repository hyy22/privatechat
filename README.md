# 私有聊天工具

> 此项目仅做学习使用

## 如何使用

1. 克隆项目

```
git clone https://github.com/hyy22/privatechat
```

2. 按需修改docker-compose配置文件

3. 开启服务

```bash
docker-compose up -d
```

## 基础功能

- [x] 注册
- [x] 登录
- [x] 私钥导入导出、重新生成
- [x] 添加好友
- [x] 同意好友请求
- [x] 删除好友
- [x] 删除好友聊天记录
- [x] 文本消息发送
- [x] 图片消息发送
- [ ] 个人信息修改
- [ ] 群聊
- [ ] 消息通知
- [ ] pwa支持
- [ ] 聊天数据导入导出
